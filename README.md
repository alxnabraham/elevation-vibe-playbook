# Elevation AI — Git + Vercel Workflow Guide

Internal documentation for the Elevation AI team on how to use Git, GitHub, and Vercel for the EI V2 Redesign project.

## Deploying

This is a static site. Push to `main` and Vercel auto-deploys from the `public/` folder.

## Editing

All content lives in `public/index.html`. Edit that file to update the documentation.
